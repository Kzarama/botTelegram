# Bot de telegram ejemplo
## Telegram
- crear bot en telegram con el usuario bot BotFather 
```
/newbot
```
## VSCode
- iniciar .json
```
$ npm init -y
```
- instalar telegraf
```
$ npm i telegraf
```
- ejecutar bot
```
$ node bot.js
```
- ejecutar node automaticamente
```
$ npm i nodemon -D
```
- cambiar linea de test en el package por
```
"dev": "nodemon bot.js"
```
- ejecutar nodemon al guardar
```
$ npm run dev"# botTelegram"
```
